Project overview

CyberLab Quest is a self-paced portal for beginner cybersecurity learners. It starts as static HTML5 pages (Weeks 1–2), then becomes styled (Weeks 2–3) and interactive (Weeks 4–5).

  • Audience: Students and early-career learners
  • Core features: Modules, lab library, glossary, progress journal
  • Optional future: Save progress to a database
Future enhancements roadmap
  • CSS: responsive layout, theme toggle (light/dark)
  • JS: save journal entries, filter labs, completion badges
  • DB (optional): user profiles + cloud sync
