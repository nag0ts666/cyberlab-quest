Learning Path

Each module includes reading goals, mini tasks, and a lab suggestion. Later (JS), you can mark completion and filter by difficulty.

  1. Module 1 — Logs & Detection

    Goal: Understand what logs are and how alerts are created.

    • Read: Event vs alert
    • Task: Identify 5 log sources in a typical company
    • Lab: Parse sample logs (see Lab Library)

    Difficulty: 2/5

  2. Module 2 — Authentication & MFA

    Goal: Explore password attacks and MFA defenses.

    • Read: Brute force vs spraying
    • Task: Draft an MFA rollout checklist
    • Lab: Simulate login telemetry review

    Difficulty: 3/5

  3. Module 3 — Web Basics & OWASP

    Goal: Learn common web vulnerabilities at a high level.

    • Read: OWASP Top 10 overview
    • Task: Map 3 risks to CIA
    • Lab: Request/response anatomy

    Difficulty: 4/5
